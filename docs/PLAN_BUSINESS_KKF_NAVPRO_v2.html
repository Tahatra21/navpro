9.2 Executive summary (ringkas dari BRD v1.0)

KKF adalah gerbang wajib sebelum CAPEX last-mile fiber disetujui. Keputusan investasi memakai NPV > 0, XIRR > WACC (9,72% p.a.), BCR ≥ 1,23 (mandatory) / ≥ 1,08 (minimum + justifikasi), dan payback dalam durasi kontrak. Permasalahan AS-IS: file Excel terisolasi, asumsi tidak terpusat, approval kertas, tidak ada audit trail, tidak ada dashboard portofolio.

8. Risiko & mitigasi (praktis)

Risiko Dampak Mitigasi
Backfill org unit Data lama tidak punya mapping unit/segment. Script migrasi + “Unassigned bucket” sementara; admin wajib assign sebelum submit.
Approver kosong/cuti Queue macet, SLA terlewat. Delegation + eskalasi otomatis; notifikasi multi-channel.
Perubahan status breaking FE/BE mismatch & data tidak terbaca. Compat layer sementara (map status lama→baru) + migrasi bertahap.
Perhitungan hari kerja SLA tidak akurat. Gunakan kalender kerja (awal: Senin–Jumat) + opsi hari libur (iterasi berikut).
